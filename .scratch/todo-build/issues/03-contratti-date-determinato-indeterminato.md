# 03 — Contratti: date determinato / indeterminato

**What to build:** Per Contratto a tempo indeterminato la data di fine non è visibile (stato “in corso”); per tempo determinato compaiono inizio e fine. Tabella e form sono allineati (oggi sembrano invertiti).

**Blocked by:** None — can start immediately.

**Status:** ready-for-agent

- [ ] Contratto OpenEnded: fine nascosta / mostrata come in corso; create/edit non richiedono data fine
- [ ] Contratto FixedTerm: inizio e fine obbligatori e visibili
- [ ] Lista Contratti e form create/edit concordano sullo stesso comportamento
